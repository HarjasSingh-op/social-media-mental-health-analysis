# Mental Health & Social Media Balance Analysis

This project explores the relationship between social media usage habits and mental well-being. Using a dataset of 500+ users, the project analyzes factors like screen time, sleep quality, and stress levels to predict an individual's overall Happiness Index.

## 📊 Project Overview
The goal of this analysis is to identify key patterns in how different social media platforms affect user stress and happiness. I implemented a **Linear Regression model** to predict happiness based on several lifestyle metrics.

## 🛠️ Tech Stack
* **Language:** Python
* **Libraries:** Pandas, Scikit-Learn, NumPy, Matplotlib/Seaborn
* **Environment:** Jupyter Notebook

## 🧬 Dataset Features
The dataset includes:
* **Usage Metrics:** Daily Screen Time, Social Media Platform.
* **Well-being Metrics:** Sleep Quality (1-10), Stress Level (1-10), Happiness Index (1-10).
* **Lifestyle Factors:** Exercise Frequency, Days without Social Media.
